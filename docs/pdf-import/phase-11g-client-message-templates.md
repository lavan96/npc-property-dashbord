# PDF Import Client Message Templates

Generic, client-safe message templates. Do not add internal diagnostics, logs,
signed URLs, storage paths, or raw content.

## 1. Template accepted

"Your template import passed our quality review and is ready for use."

## 2. Template accepted with warnings

"Your template import has been accepted. We noted minor layout warnings that do
not affect usability."

## 3. Manual review required

"Your template is undergoing a manual quality review before approval. We will
confirm shortly."

## 4. Import rejected / rework required

"The import did not meet our quality requirements and needs rework. We will
follow up with next steps."

## 5. Export validation pending

"Export validation is pending final checks before we confirm readiness."

## 6. Quality review in progress

"Your template is currently in our quality review process."
