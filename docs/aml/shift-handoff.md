# AML Shift Handoff Template

Post this at end-of-shift into the AML on-call channel.

```
AML shift handoff — [YYYY-MM-DD, AM/PM]
Analyst: [name]

Open items:
- [Case ID] [status] [next action + owner + due]
- ...

New matches this shift:
- [count] sanctions, [count] PEP, [count] adverse-media
- [count] auto-cleared, [count] escalated to EDD

Reports:
- SMRs drafted: [n]
- SMRs signed off by MLRO: [n]
- TTR / IFTI due: [next date]

Alerts overnight/day: [count acknowledged, count escalated]

Provider health:
- [provider] [status] [any incidents]

Risks / blockers:
- [describe or write "none"]

Next-shift focus:
- [3 bullets max]
```

Log the same handoff summary as a note on the highest-priority open case, so it lives with the audit trail.
